安装Hexo博客

Hexo博客的发布的整体流程为：

1. 本地撰写博客 
2. git push到Github
3. Github Web Hooks触发请求博客部署的服务器
4. 服务器收到Github Web Hooks的请求后执行`git pull`命令并生成Hexo静态博客

部署本地Hexo环境

* 根据Hexo官网教程安装本地环境：

* 创建Github Repository用于博客同步。这里创建一个名为blog的 [GitHub - ylm201/myblog](https://github.com/ylm201/myblog)
* 安装 [hexo-deployer-git](https://github.com/hexojs/hexo-deployer-git)

* 部署Hexo：执行`hexo g && hexo d`生成并部署博客到Github Repo。如果碰到命令行提示`Cannot find module ‘./build/Release/DTraceProviderBindings`可以通过执行`npm install hexo —no-optional`解决。(参考 [Github issue](https://github.com/hexojs/hexo/issues/1326#issuecomment-113871796) ）
